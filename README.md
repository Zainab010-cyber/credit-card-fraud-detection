# credit-card-fraud-detection
# Explainable Machine Learning for Credit Card Fraud Detection

## Project Overview
This project applies machine learning techniques to detect fraudulent credit card transactions using SMOTE and ensemble learning methods.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Techniques Used
- Data preprocessing
- Feature scaling
- SMOTE oversampling
- Hyperparameter tuning
- Cross-validation
- Explainable AI (SHAP)
- Threshold optimisation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

## Project Outputs
- Confusion matrix
- ROC curve
- Precision-recall curve
- Feature importance
- SHAP analysis
- Error analysis

## How to Run
1. Open the notebook in Google Colab.
2. Upload creditcard.csv dataset.
3. Run all cells sequentially.
4. View generated figures and results.

## Dataset
Kaggle Credit Card Fraud Detection Dataset
